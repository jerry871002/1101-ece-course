# Cheatsheet

## 1. Headers

# This is an \<h1> tag
## This is an \<h2> tag
###### This is an \<h6> tag

## 2. Emphasis

*This text will be italic*<br>
_This will also be italic_

**This text will be bold**<br>
__This will also be bold__

_You **can** combine them_

### Unordered

### Ordered
1. Item 1
1. Item 2
